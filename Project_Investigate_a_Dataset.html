Introduction¶

Dataset Description¶

This project useses tmdb-movies dataset. This dataset has a shape of 10866 rows and 21 columns containing information about movies released from 1960 to 2015. Information of a film has the following attributes:

id                      int64   - A unique identifier for each movie
imdb_id                 object  - A unique identifier for each movie on IMDB.
popularity              float64 - A numeric quantity specifying the movie popularity.
budget                  int64   - The budget in which the movie was made.
revenue                 int64   - The worldwide revenue generated by the movie.
original_title          object  - The title of the movie before translation or adaptation.
cast                    object  - The name of lead and supporting actors.
homepage                object  - A link to the homepage of the movie.
director                object  - The director of the movie
tagline                 object  - Movie's tagline.
keywords                object  - The keywords or tags related to the movie
overview                object  - A brief description of the movie
runtime                 int64   - The running time of the movie in minutes.
genres                  object  - The genre of the movie, Action, Comedy ,Thriller etc.
production_companies    object  - The production house of the movie.
release_date            object  - The date on which it was released. This column need to be converted into datetime
vote_count              int64   - The number of votes
vote_average            float64 - Average ratings the movie recieved
release_year            int64   - The year of movie released
budget_adj              float64 - The budget associated movie
revenue_adj             float64 - the revenue associated movie

def get_data(x):
    df_data = df[x].str.cat(sep = '|')
    df_data = pd.Series(df_data.split('|'))
    df_data = df_data.value_counts(ascending=False)
    return df_data

Conclusions¶

The TMDb dataset is very useful in information.¶

- From the point of view a movie watcher, you may know which film is worth seeing and recommend to your friends bassing on vote_average and popularity factors.¶

- From the point of view of an film producer, you can easy decide which genre to make a new film and you can know what movie has the high popularity tends to earn high profit.¶

- From the point of view of an investor, you can find what movie has the highest revenue and the top movies has the highest profit.¶

Limitations:¶

Beside the good points, the dataset has some down points need to be avoid to improve quality of the dataset such as:¶

- Missing values in some comlumns affecte the overall analysis.¶

- Data in some columns are not good leading to wrong analysis.¶

For example: budget and revenue columns do not have currency unit, the gap between movies are too big. It is similar to runtime column.
