Real-world Data Wrangling¶

1. Gather data¶

1.1. Problem Statement¶

In this project I will use Coronavirus (Covid-19) Data in the United States datasets. The data are the daily cumulative number of cases and deaths reported in each county and state across the U.S. since the beginning of the pandemic.

The data in these files is a different version of the data in our main U.S. cases and deaths files. Instead of cumulative totals, each file contains the daily number of new cases and deaths, the seven-day rolling average and the seven-day rolling average per 100,000 residents.

https://github.com/nytimes/covid-19-data

Dataset 1: Anomalies¶

Type: csv

Method: Read_csv() - the file was downloaded from website

Dataset variables:

date: The date to which the anomaly applies.
end_date: The last day to which it applies. Otherwise, left blank.
geoid: A unique geographic identifier for each place.
type: Whether the anomaly applies to the data on cases, deaths or both.
omit_from_rolling_average: This will be yes if the data for that day is excluded from the calculation of rolling averages. Otherwise, left blank.
adjusted_daily_count_for_avg: If the daily total includes a large known backlog.
description: A note explaining the cause for the anomaly.

Dataset 2: Rolling Averages¶

Type: CSV

Method: Download from website via URL

Dataset variables:

geoid: A unique geographic identifier for each place.
cases: The number of new cases of Covid-19 reported that day.
cases_avg: The average number of new cases reported.

deaths: The total number of new deaths from Covid-19 reported that day.
deaths_avg: The daily average number of new deaths reported.
deaths_avg_per_100k: The deaths_avg per 100,000 people.
